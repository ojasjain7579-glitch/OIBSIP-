# Number Guessing Game

## Project Description

This is a simple Number Guessing Game developed using Java, HTML, and CSS.

In this game, the computer generates a random number between 1 and 100 and the user has to guess the correct number.

The game gives hints like Too High or Too Low until the user guesses the correct number.


## Features

- Random number generation
- User input
- Multiple rounds
- Score calculation
- Higher and lower hints
- Simple user interface


## Technologies Used

- Java
- HTML
- CSS
- VS Code

## How To Run Project

Step 1: Open project folder in VS Code.


Step 2: Run Backend

Open terminal:

cd backend

Compile:

javac NumberGuessingGame.java

Run:

java NumberGuessingGame


Step 3: Run Frontend

Open frontend folder.

Open index.html.

Right click on index.html.

Click Open with Live Server.


## How To Play

1. Start the game.

2. Enter a number between 1 and 100.

3. If the guess is wrong, the game shows:
   - Too High
   - Too Low

4. Guess the correct number and get score.
